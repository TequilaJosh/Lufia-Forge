# Lufia-Forge
⚔️ Lufia Forge — A C# WPF ROM hack toolkit for Lufia &amp; The Fortress of Doom (SNES). Features an embedded Snes9x emulator with live RAM/VRAM inspection, 65816 disassembler, dialogue editor, tile viewer, map maker, script/event editor, and character editor. Research while you play. Edit everything.
